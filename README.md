# DOBBY

1.프로젝트 수행 목적

1.1 프로젝트 정의

 - Firebase를 활용한 멀티 플랫폼 출퇴근 어플

1.2 프로젝트 배경

  - 기존에 사용하던 구글 시트 출퇴근 프로그램이 모바일 환경에선 작동하지 않아 Firebase를 활용하여 출퇴근 프로그램을 개발

1.3 프로젝트 목표
   
  - 로그인/로그아웃
  - 출퇴근 기능
  - Android, IOS 간 출퇴근 기록 공유 및 조회
  - 실시간 시간 조회
